    Design Source

    RIFTBLADE — built to the v1.0 MVP specification. Two bosses, one blade, no mercy.

    Code

    Original engine & game code written for this build (HTML5 Canvas + Web Audio). Fixed-timestep 60 Hz simulation. Zero network calls; runs fully offline.

    Art

    All characters, bosses, arenas, telegraphs and UI are original procedural vector art drawn at runtime. No third-party assets are included. (Policy A-01 satisfied: nothing to attribute, nothing encumbered.)

    Audio

    All sound effects and music are synthesized at runtime with the Web Audio API. Original composition; no recorded samples.

    Typography

    System font stacks only (serif/system-ui) — no embedded or fetched fonts, keeping the build dependency-free and offline.
